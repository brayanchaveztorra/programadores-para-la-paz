En otra terminal, envié la petición:
curl -X POST http://localhost:3000/registro -H "Content-Type: application/json" -d '{"nombre":"Brayan","mensaje":"Hola comunidad y mundo"}'

Respuesta del servidor:
{"estado":"Datos recibidos","nombre":"Brayan","mensaje":"Hola comunidad y mundo"}

El servidor recibe los datos enviados en el cuerpo JSON y devuelve un objeto JSON con el estado y los mismos valores que yo envié.
